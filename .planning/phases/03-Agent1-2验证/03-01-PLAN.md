**步骤 2：改造 agent1_preprocessor.py**

完整重写 agent1_preprocessor.py，移除 anthropic 和 PaddleOCR，改为 GLM-4.6V + Qwen3.5 双路：

- 移除 `import anthropic` 和 `from utils.ocr import extract_text_from_image`
- 新增 `import requests` 和 `from openai import OpenAI`
- `__init__` 方法：
  - 从环境变量 `ZHIPU_API_KEY` 初始化 GLM 客户端（使用 requests，base_url: `https://open.bigmodel.cn/api/paas/v4`）
  - 从环境变量 `NAU_API_KEY` 初始化 Qwen 客户端（使用 OpenAI SDK，base_url: `https://ai-api.nau.edu.cn/v1`）
  - 从 `config/intent_labels.json` 读取合法标签列表，存入 `self.valid_labels`（id 字符串列表）
- `run()` 方法主流程改为两步：
  1. **GLM 图像理解**（仅 image 类型）：将图片 base64 编码后，通过 requests.post 发送到 `https://open.bigmodel.cn/api/paas/v4/chat/completions`，model="glm-4.6v"，请求体含 `{"role": "user", "content": [{"type": "image_url", "image_url": {"url": "data:{media_type};base64,{img_data}"}}, {"type": "text", "text": "请详细描述这张图片的内容，包括所有文字、图形、颜色和可能的含义。"}]}`，超时 60 秒，获得图像描述文本 `image_description`。邮件类型跳过此步，直接用 body_text 作为内容。
  2. **Qwen 意图分类**：将 `image_description`（或邮件文本）拼入提示词，调用 Qwen3.5（`client.chat.completions.create(model="qwen3.5-35b-a3b-fp8", ...)`），system prompt 包含 SYSTEM_PROMPT，要求输出 JSON 格式含 `intent_label`、`confidence`、`key_indicators`、`extracted_text_summary`、`reasoning`。
- 保留 `SYSTEM_PROMPT` 类变量（更新意图标签说明为从配置读取）
- 输出字段改名以匹配规格：`key_indicators` → 保留原名，`extracted_text_summary` → 保留原名
- 删除所有 `INTENT_LABELS` 类变量（改从配置文件读取）
- 保留 step_dir / step1.json 写入逻辑不变
- 保留 try/except 错误处理结构不变

GLM API 调用代码模式（来自 api-key-hardcode-example）：
```python
resp = requests.post(
    f"{glm_base_url}/chat/completions",
    headers={"Authorization": f"Bearer {glm_api_key}", "Content-Type": "application/json"},
    json={"model": "glm-4.6v", "messages": [...], "max_tokens": 1024},
    timeout=60,
)
resp.raise_for_status()
glm_result = resp.json()
image_description = glm_result["choices"][0]["message"]["content"]
```

Qwen API 调用代码模式：
```python
qwen_client = OpenAI(base_url="https://ai-api.nau.edu.cn/v1", api_key=nau_api_key)
response = qwen_client.chat.completions.create(
    model="qwen3.5-35b-a3b-fp8",
    messages=[{"role": "system", "content": SYSTEM_PROMPT}, {"role": "user", "content": analysis_prompt}],
    max_tokens=1024,
)
response_text = response.choices[0].message.content.strip()
```

注意：若 `ZHIPU_API_KEY` 或 `NAU_API_KEY` 未设置，在 `__init__` 中抛出 `EnvironmentError` 并给出明确提示。
