- 返回 list[dict]，每个 dict 形如：


- 日志：`logger.info(f"[Agent4] 从 capabilities.md 加载 {len(rules)} 条历史规则")`

3. 在 run() 的"规则引擎调整"段（步骤 2，第 69 行附近），在 for rule in self.rules... 循环之后，追加 memory rules 处理：
```python
# 2b. 历史学习规则调整（来自 capabilities.md）
for mem_rule in self.memory_rules:
    mem_keywords = mem_rule.get("keywords", [])
    mem_weight = mem_rule.get("weight", 0.5)
    matched = [kw for kw in mem_keywords if kw in extracted_text]
    if matched:
        adjustment = mem_weight * (len(matched) / max(len(mem_keywords), 1)) * 0.1
        rule_adjustments.append({
            "rule_id": "memory_rule",
            "rule_name": "历史学习规则",
            "matched_keywords": matched,
            "adjustment": adjustment
        })
```
