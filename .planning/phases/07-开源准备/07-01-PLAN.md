# AntiScam Agent System — 智能反诈骗分析系统

## 项目简介
简短描述：基于多 Agent 架构的自动反诈骗分析系统，能自动检测诈骗内容并通过自我迭代学习持续改进判断能力。

## 系统架构
5-Agent 流水线说明（一段文字 + ASCII 流程图）：
```
图片/文本 → Agent1(预处理+OCR) → Agent2(案例检索) → Agent3(诈骗判别) → Agent4(风险评估) → Agent5(干预通知)
                                                                                    ↑
                                                                          ReflectorAgent(误判反思)
```

## 快速开始

### 环境要求
- Python 3.10+
- Windows 11（PaddleOCR 测试环境）
- 说明 Visual C++ Redistributable 依赖

### 1. 克隆仓库
```bash
git clone <repo-url>
cd mutiagent_trea
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```
注意事项：PaddleOCR 在 Windows 若安装失败，执行 `pip install "paddleocr>=2.7.0" --no-deps` 后手动安装依赖。

### 3. 配置环境变量
```bash
cp .env.example .env
```
编辑 .env 文件，填入以下必需变量：
| 变量名 | 说明 | 获取方式 |
|--------|------|----------|
| ZHIPU_API_KEY | 智谱 AI API Key（GLM-4.6V 图片理解） | https://open.bigmodel.cn |
| NAU_API_KEY | Qwen 兼容 API Key（意图识别） | 联系提供商 |
| NAU_BASE_URL | Qwen API Base URL | 联系提供商 |
| GUARDIAN_FEISHU | 飞书 Webhook URL（干预通知，可选） | 飞书开发者平台 |
| EMAIL_USER | 邮件监控账号（可选） | 邮件服务商 |
| EMAIL_PASS | 邮件监控授权码（可选） | 邮件服务商 |

### 4. 初始化案例库
```bash
python -m scripts.init_lancedb
```
预期输出：`案例库初始化成功，共 8 条记录`

### 5. 运行测试
```bash
pytest tests/ -v
```
预期：121 个测试全部通过。

### 6. 启动监控
```bash
# WeChat 文件夹监控
python -m monitor.wechat_monitor

# 邮件监控（需配置 EMAIL_USER/EMAIL_PASS）
python -m monitor.email_monitor
```

### 7. 手动触发分析（单次测试）
```bash
python -c "
from pipeline.orchestrator import Orchestrator
import asyncio
orch = Orchestrator()
result = asyncio.run(orch.run('cases/raw/test.jpg'))
print(result)
"
```

```bash
python -m memory.reflector --task-id <task-id> --feedback "误判原因"
```
反思结果写入 `memory/reflections/` 并更新 `memory/capabilities.md`，影响下次分析。

### Prerequisites
- Python 3.10+
